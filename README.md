Clone the code into your desired IDE

Open the terminal of the IDE, it should be on this project

Type "cd client" - if you get some kind of error that means the terminal is not on this project

Now it should be inside the client folder

Type "npm install" a bunch of files should load in, don't worry if theres errors or some files are not loaded in as long as it says it finished

Type "npm init" some more stuff should load

Finally type "npm start" this should run the code in the client folder, it should automatically pop up in google but if not, try googling http://localhost:3000/home and seeing if it pops up

right now its just the bar when it first loads just hit home and it should take you to the website
